## GithubTask
## this app gets data of users from [ Github api](https://docs.github.com/en/rest/reference/search#search-topics) and displays info about each user when tapped

## Tools :
       
    * Kotlin
    * MVVM
    * Retrofit
    * Paging 3
    * viewModel , LiveData
    * Navigation component 
    * Hilt
    * ViewBinding
    * RecyclerView .. DiffUtil
